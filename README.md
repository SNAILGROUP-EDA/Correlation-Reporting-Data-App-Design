> 많은 자료를 이용하여 다양한 시선에서 분석하고, 깔끔하게 시각화 한 점이 좋았습니다.

> 데이터프레임을 구체적으로 잘 만들었습니다. 시각화를 염두에 두고 데이터프레임을 가공하는 상세한 과정이 좋았습니다. 또한 시각화에서 그치지 않고 인사이트를 도출한 점이 인상 깊었습니다.

> 여러 방향으로 자료를 탐색했고, 그 결과를 청중이 이해하기 수월하도록 깔끔하게 정리한 점이 좋았습니다.

</details>
